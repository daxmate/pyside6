# PySide6 Learning Project

A PySide6 learning project demonstrating custom QListWidget styling with Qt Style Sheets (QSS).

## Features

- Custom QListWidget implementation with word wrap support
- Modern styled UI using Qt Style Sheets
- Light theme color scheme

## Project Structure

```
pyside6/
├── main.py                 # Main application entry
├── dlistwidget.py          # Custom QListWidget subclass
├── forms/
│   └── mainwindow_ui.py    # UI generated by Qt Designer
└── styles/
    └── listwidget.qss      # QSS stylesheet for QListWidget
```

## Requirements

- Python 3.10+
- PySide6

## Installation

```bash
# Create virtual environment
python -m venv .venv

# Activate virtual environment
# macOS/Linux
source .venv/bin/activate

# Windows
.venv\Scripts\activate

# Install dependencies
pip install pyside6
```

## Running the Application

```bash
python main.py
```

## Customizing Styles

Edit `styles/listwidget.qss` to customize the appearance of the QListWidget:

- Background color
- Border styles
- Item hover/selected states
- Scrollbar appearance
- Font settings
